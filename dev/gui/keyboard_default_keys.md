# Default keys

The following list contains all default keyboard behaviours:

| Key | Modifier 1 | Modifier 2 | Modifier 3 | Action                |
|-----|------------|------------|------------|-----------------------|
| F8  |            |            |            | Add new data          |
| F5  |            |            |            | Open data for editing |
| F7  |            |            |            | Remove data           |
| F9  |            |            |            | Save data             |
